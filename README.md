# Simple RPG

Este repositório contém um jogo de RPG simples desenvolvido para navegadores, permitindo que os jogadores explorem e interajam com o ambiente de forma dinâmica.

## Demonstração

Acesse o projeto em: [Simple RPG](https://simple-rpg-brown.vercel.app/)

## Tecnologias Utilizadas

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Nenhum (Projeto Estático)
- **Deploy:** Vercel

## Funcionalidades

- Movimento do personagem
- Interação com objetos do cenário
- Mecânica básica de combate
- Progressão de nível do personagem

## Como Executar Localmente

```sh
# Clone o repositório
git clone https://github.com/seu-usuario/simple-rpg.git

# Acesse o diretório do projeto
cd simple-rpg

# Abra o arquivo index.html em um navegador
```
